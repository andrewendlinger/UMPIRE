## Installation -- Usage

```bash
$ git clone https://github.com/DiracKeteering/UMPIRE.git

$ cd UMPIRE

$ pip install -e .
```

## Installation -- Development
```bash
$ git clone https://github.com/DiracKeteering/UMPIRE.git

$ cd UMPIRE

$ pip install -e ".[dev]"
```
